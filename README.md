# 🎬 YouTube 1080p 智能剪辑器 (Colab 版)

这是一个无需安装任何软件、利用 Google 云端算力的在线剪辑工具。

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](你的直达链接粘贴在这里)

### 使用说明：
1. 点击上方按钮进入 Colab。
2. 点击菜单栏的 **代码执行程序 (Runtime)** -> **全部运行 (Run all)**。
3. 在输出区域找到 **Password** 并复制。
4. 点击输出的 `loca.lt` 链接，输入 Password 即可进入网页界面。
